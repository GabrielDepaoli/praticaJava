# Objetivo
Neste Projeto, quero inserir todos os meus conhecimentos e aprendizados no meu primeiro ano de faculdade, além de buscar desenvolver novas funcionalidades! 
Este projeto se inicia em 11/11/2025, e não tenho data de finalização: meu principal objetivo não é terminá-lo, mas colocar na 'mesa' tudo que aprendi até aqui!

Aqui vou focar no desenvolvimento de Java Swing, trazendo diversas funcionalidades como:
- Lançamentos de valores(entrada e saída)
- Listagem de Movimentações
- Gastos Mensais
- Cálculo automático de saldo
